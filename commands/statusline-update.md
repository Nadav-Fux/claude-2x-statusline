---

Update the user's installed claude-2x-statusline.

## Steps

1. Prefer the installed updater at `~/.claude/cc-2x-statusline/update.sh`.
2. On Windows without bash, use `powershell -NoProfile -ExecutionPolicy Bypass -File "$env:USERPROFILE\.claude\cc-2x-statusline\update.ps1"`.
3. Show the updater output verbatim.
4. Tell the user to restart Claude Code when the update finishes.
